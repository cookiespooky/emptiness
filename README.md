# Emptiness

Интерактивная визуализация масштаба космической пустоты. Полностью статический проект без бэкенда и внешних зависимостей.

## Запуск локально

Подойдёт любой статический сервер:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## GitHub Pages

Деплой выполняется workflow `.github/workflows/pages.yml` после каждого пуша в `main`.

В настройках репозитория выберите:

`Settings → Pages → Source → GitHub Actions`

После первого успешного workflow сайт будет доступен по адресу:

`https://cookiespooky.github.io/emptiness/`

## Мобильная адаптация

Мобильный браузер считается основным поддерживаемым сценарием:

- управление вертикальным свайпом;
- поддержка safe-area на iPhone;
- адаптация к портретной и альбомной ориентации;
- Canvas с учётом плотности пикселей устройства;
- элементы управления не меньше 44 px;
- сокращённая анимация при `prefers-reduced-motion`;
- адаптивные подписи, HUD и типографика;
- отсутствие горизонтального скролла и тяжёлых зависимостей.
